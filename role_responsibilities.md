# Team Roles & Responsibilities (2-Person Team)

## Team Structure
The project is developed by a 2-person team with clear separation of responsibilities to ensure efficiency and accountability.

---

## Role 1: Frontend & UX Engineer

### Primary Responsibilities
- Design and implement the Chrome extension UI
- Develop:
  - Floating action button
  - Overlay clean view
  - Loading and state transitions
- Implement DOM manipulation:
  - Page text extraction
  - Button/link detection
  - Element highlighting and scrolling
- Ensure accessibility-focused UI:
  - Large fonts
  - Clear spacing
  - Simple layouts
- Manage extension configuration:
  - `manifest.json`
  - Permissions and content scripts

### Key Deliverables
- Fully functional Chrome extension
- Clean and intuitive user interaction flow
- Robust handling across different websites

---

## Role 2: Backend & AI Engineer

### Primary Responsibilities
- Design and deploy Python backend on Vercel
- Integrate Gemini API securely
- Develop AI prompt strategies for:
  - TL;DR summarization
  - Clean content rewriting
  - Goal action reasoning
- Ensure:
  - Strict JSON responses
  - Low-latency calls
  - Reliable error handling
- Implement CORS handling for extension communication

### Key Deliverables
- Deployed backend endpoints
- Stable and predictable AI responses
- Secure API key management

---

## Shared Responsibilities
- System architecture decisions
- UX flow planning
- Feature prioritization
- Testing on real-world websites
- Debugging integration issues
- Preparing demo and presentation materials

---

## Collaboration Workflow
- Regular sync on interface contracts (API payloads)
- Clear ownership of components
- Shared repository with documented code
- Rapid iteration based on testing feedback
